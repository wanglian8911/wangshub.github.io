python 装饰器学习笔记

概括

装饰器的作用就是为已经存在的对象添加额外的功能

我们可以继续调用装饰器来修饰函数，而不用重复修改函数或者增加新的封装。这样就提高了程序的可重复利用性，并增加了程序的可读性。
